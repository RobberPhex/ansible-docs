dladm_etherstub - Manage etherstubs on Solaris/illumos systems.
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. versionadded:: 2.2


.. contents::
   :local:
   :depth: 2


Synopsis
--------

* Create or delete etherstubs on Solaris/illumos systems.

Examples
--------

 ::

    # Create 'stub0' etherstub
    - dladm_etherstub:
        name: stub0
        state: present
    
    # Remove 'stub0 etherstub
    - dladm_etherstub:
        name: stub0
        state: absent
