.. _group:


group - Add or remove groups
++++++++++++++++++++++++++++



.. contents::
   :local:
   :depth: 1


Synopsis
--------

Manage presence of groups on a host.


Requirements
------------

  * groupadd
  * groupdel
  * groupmod
