nxos_file_copy - Copy a file to a remote NXOS device over SCP.
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. versionadded:: 2.2


.. contents::
   :local:
   :depth: 1


Synopsis
--------

Copy a file to the flash (or bootflash) remote network device on NXOS devices.

Notes
-----

.. note:: The feature must be enabled with feature scp-server.
.. note:: If the file is already present (md5 sums match), no transfer will take place.
.. note:: Check mode will tell you if the file would be copied.
